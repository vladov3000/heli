# heli

## Dependencies

Rust cargo

## Installation

    git clone https://github.com/vladov3000/heli.git && cd heli
    chmod +x install.sh
    ./install.sh

## Uninstall

    # In heli dir
    chmod +x uninstall.sh
    ./uninstall.sh
    
## Usage

    heli
